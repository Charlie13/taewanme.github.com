1.1 오라클 클라우드 계정 확보

현재 사용 가능한 Oracle Cloud 계정이 있는 상태라면, 이 페이지는 건너 뛰셔도 됩니다. 다음 페이지로 이동하시기 바랍니다.

Oracle Autonomous Data Warehouse Cloud Service(ADWC) 서비스를 만들기 위해서는 우선 Oracle Cloud 계정 있어야 합니다. Oracle Cloud Trial은 오라클 클라우드를 알리기 위한 프로모션 서비스입니다. 사용자에게 30일간 300불 크래딧을 무료로 제공하는 합니다. 사용자는 이 기간 동안 할당된 크래딧을 제약 없이 사용할 수 있습니다. 이 프로그램에서 자원 사용량은 시간 단위로 계산되고, 300불 크래딧에서 차감됩니다. 300불 크래딧이 모두 소진되거나 1달 기한이 넘어가면 오라클 클라우드 트라이얼 프로그램은 종료됩니다.

현재 이용 가능한 Oracle Cloud 계정이 없다면, Oracle Cloud Trial 계정을 생성하시기 바랍니다.

오라클 클라우드 트라이얼 신청하기 위해서는 신용카드(Visa 카드와 Master 카드), 핸드폰 번포 및 E-mail를 입력해야 합니다. 3가지 정보를 사용자 인증에만 사용합니다. 특히 신용카드는 사용자 확인 절차일 뿐이며, 과금을 목적으로 사용되지 않습니다. 사용자 확인 목적으로 1불을 결제하고 1분 후에 해당 결제를 취소합니다. 오라클 클라우드 트라이얼은 사용 기간과 크래딧이 한정된 서비스입니다. 30일 동안 제공된 크래딧을 초과하여 사용할 수 없습니다. 따라서 사용량 초과에 대한 신용카드 청구는 발생하지 않습니다.

현재 이용 가능한 Oracle Cloud 계정이 없기 깨문에, Oracle Cloud Trial을 신청해야 한다면 다음 동영상, Oracle Cloud Trial 신청 가이드 문서를 참조하여 신청하시기 바랍니다.

  • Oracle Cloud Trial 신청 가이드 문서[↗NW]
    • 스크린 샷 중심으로 Oracle Cloud Trial 신청 과정을 설명합니다.
  • Oracle Cloud Trial 등록 동영상[↗NW]
    • Oracle Cloud Trial 신청 동영상입니다.
    • 이 동영상은 한국어로 Oracle Cloud Trial를 한국말로 진행합니다.

신청 과정에서 발생하는 주요 질문과 문의 사항은 다음 링크에서 확인할 수 있습니다.

  • Oracle Cloud Trial 신청 FAQ[↗NW]

Oracle Cloud Trial 선청 후, 계정 활성화까지 약 15분 ~ 30분 정도 걸립니다. Oracle Cloud Trial를 이용하여 오라클 클라우드 리전별로 8 OCPU ADW를 생성할 수 있습니다. 1 OCPU1가 할당된 ADW 인스턴스 8개를 만들수 도 있고, 8 OCPU로 1개 인스턴스를 만들 수도 있습니다.

참고문서

  • OCPU 설명: https://docs.oracle.com/en/cloud/iaas/compute-iaas-cloud/stcsg/shapes.html[↗NW]
  1. OCPU는 Oracle Compute Unit의 약자로 Oracle Cloud의 CPU 단위 입니다. OCPU는 물리적인 1개 프로세스입니다. 1개 OCPU는 두 개의 하드웨어 실행 스레드에 해당합니다. 따라서 1개 OCPU는 2 vCPU(AWS)와 같습니다. [return]
